## 装配说明

本项目涉及的五金件可能包括但不限于：
- 608ZZ 轴承
- M3*2.5 热熔通孔螺母
- M4 沉头螺丝
- M3 杯头螺丝

由于规格组合较多，不在此维护完整清单。装配与下料请以 CAD/打印工程为准。
