## Assembly Notes

This project may involve, but is not limited to:
- 608ZZ bearings
- M3*2.5 heat-set inserts
- M4 countersunk screws
- M3 socket head screws

Because combinations vary, no full BOM is maintained here. Use CAD and printing project files as the final reference for assembly.
